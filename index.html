decroghlangs

Decroghlangs is a fetching tool that pulls data about programming languages usage in one's public GitHub repositories.

Fill in the form below, then click the button in order to get your desired data.

The results will appear here once they are loaded...

Important: if you don't see your results, here are some reasons why:

  • The username is wrong;
  • You have sent too many requests (wait some time before continuing);
  • The user simply does not exist;
  • The user does not have public repositories;

Important: if your results seem to be inaccurate, check if you have a lot of repositories. Currently, this tool will only check your 50 most recent ones.
