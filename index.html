decroghlangs

Decroghlangs is a fetching tool that pulls data about programming languages usage in one's public GitHub repositories.

Fill in the form below, then click the button in order to get your desired data.

The results will appear here once they are loaded...

I don't see my results!

If you don't see your results, here are some precautions to take:

  • Check the username's spelling;
  • Check whether the user actually exists;
  • Make sure the user has public repositories;
  • Wait awhile and try again later.

My results seem to be inaccurate!

In case your results seem to be inaccurate, check if the user has many repositories. Currently, this tool will only check the 50 most recent ones.
